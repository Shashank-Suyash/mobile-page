LDR BASED LIGHTING SYSTEM

This project is a lighting system based on Light Dependent Resistors (LDR). It automatically adjusts the lighting depending on the surrounding light levels, providing energy-efficient lighting solutions for homes and offices.

FIRE ALARM

This fire alarm system uses a smoke detector to monitor for fire hazards. Once smoke is detected, an alarm is triggered to alert individuals in the vicinity, potentially preventing fire damage.

Voice Activated LED

This project demonstrates voice control of an LED light. Using a voice recognition system, the LED can be turned on or off based on spoken commands, adding convenience and accessibility.

Smoke Detector

This smoke detector system senses the presence of smoke particles in the air and immediately triggers an alarm to warn people about a potential fire hazard.

Smart Irrigation

The smart irrigation system uses sensors to monitor soil moisture levels and adjust watering schedules accordingly. It helps conserve water while ensuring plants receive adequate hydration.

Smart Authorisation Using RFID

This RFID-based smart authorization system uses radio-frequency identification to grant access to secure areas. It allows for efficient and secure authentication, ideal for applications such as access control systems.
